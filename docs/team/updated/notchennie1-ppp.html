Project: ModuleSync

ModuleSync is a desktop CLI-based task manager that helps university students manage module-related tasks across semesters. It is optimized for users who prefer fast command-driven workflows over GUI interactions.

Overview

I focused on three main areas: task filtering/deletion for day-to-day productivity, semester-level visibility features, and semester lifecycle controls with read-only safety.
